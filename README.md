# Vulnerable application

## Setup

The application is developed and tested with:

- Node.js v23.11.0
- npm v11.3.0

### Installation and Running

1. Clone the repository

2. Install dependencies with npm

   ```bash
   npm install
   ```

3. Start the application

   ```bash
   npm run serve
    ```
